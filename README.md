# PromoBot - Telegram Points & Promotion Bot

A complete Telegram bot for points-based task completion and content promotion.

## Features

- Task System - Complete tasks to earn points
- Points Management - Earn, spend, and track points
- Promotion System - Promote your content using points
- Referral Program - Earn points by inviting friends
- Daily Bonus - Get free points every day
- Telegram Stars - Buy points with Telegram Stars
- Admin Dashboard - Broadcast, stats, backup
